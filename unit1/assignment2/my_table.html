Question 1: What are the basic parts of a table, and which tags identify them?

The table includes 3 parts, the table as a whole is one part, each row is a part, and each column in a row is also a part.

The table part uses the table tag, the row part uses the tr tag and the column part uses the td tag.

Question 2: Which attribute is the most common attribute in the table tag and what does it do?

The border attribute is the most common attribute because every table usually has a border and this attribute defines how the border should look (width, color, and style).

Which attributes are used to create cells that span more than one column or row?

The attributes colspan and rowspan allow a cell to extend into another column or row.

Which elements are used to define the head, body, and foot of a table?

thead defines the head, tbody defines the body, and tfoot defines the foot of a table
